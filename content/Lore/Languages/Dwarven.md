# Speakers
Dwarven is the language spoken and read by Dwarves. 

# Script
Dwarven script can be written in traditional or simplified script, with traditional script containing many serifs and calligraphic curls for embellishment, and simplified script being solely straight lines for ease of carving.

# Phonics
Dwarven is a deep, resonant language that echoes for long periods in caverns and large spaces.
